# VIT-LEARNS
VIT Learns — Online Course Platform

A beginner-friendly Online Course Platform Website built using HTML, CSS, and JavaScript.
This project demonstrates core front-end development concepts like DOM manipulation, form validation, filtering, and LocalStorage.

🌐 Live Demo

Deployed Link: https://kushalggoyal.github.io/VIT-LEARNS/


💻 GitHub Repository

Source Code: https://github.com/KushalGGoyal/VIT-LEARNS

📌 Project Overview

VIT Learns is a simple learning platform where users can:

Browse available courses
Filter courses by category
View course curriculum
Enroll in a course
Save enrolled courses in the browser
View enrolled courses in a dashboard

This project was created as a mini project for learning core web development.

✨ Features
🏠 Homepage
Modern hero section
Navigation bar (Courses, Enroll, Dashboard)
Smooth scrolling navigation
📚 Course Catalog
Course cards with title, description & category
Filter courses using JavaScript
Expand / Collapse course modules (accordion)
📝 Enrollment Form
Input validation:
Name validation
Email validation
Course selection validation
Success message after enrollment
💾 LocalStorage Integration
Saves enrolled courses in browser storage
Dashboard shows saved courses
Clear all enrollments button

⚙️ How It Works
1️⃣ Course Filtering

JavaScript filters courses based on selected category using data attributes and DOM manipulation.

2️⃣ Accordion Modules

Course curriculum expands and collapses using event listeners.

3️⃣ Form Validation

JavaScript checks:

Empty name field
Valid email format
Course selected
4️⃣ LocalStorage

When a user enrolls:

Course is saved in LocalStorage
Dashboard loads saved courses automatically
